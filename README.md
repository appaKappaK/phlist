# Pi-hole Combined Blocklist Generator

A Python tool that fetches, parses, and deduplicates multiple Pi-hole blocklists into a single optimized list.

## Features

- Fetches multiple blocklists from various sources
- Parses and validates domain entries
- Deduplicates entries for optimal performance
- Outputs a clean, combined blocklist for Pi-hole
- Handles different list formats automatically

## Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/PiHoleCombineList.git
cd PiHoleCombineList
```
